# Digital Marketplace System
It is a C++ application for managing digital assets and items. It is very much inspired by the Steam marketplace and includes features like rarity, item condition (float), and prices.

# Features:
- Dynamic Inventory Management: Add, display, and track items with automatic capacity expansion.
- Rarity System: Items are assigned different rarities based on odds (configurable)
- Wear Condition/Float system: Just like CS items, each item will be assigned a condition randomly
- Dynamic pricing system: Based on the Condition and the rarity, each item is given its own dynamic pricing
- Report generation: Easily generate a report as a text file with ease.
